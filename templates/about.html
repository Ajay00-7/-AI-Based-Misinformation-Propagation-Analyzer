Project Overview

The AI-Based Misinformation Propagation Analyzer is a production-grade, enterprise-ready platform designed to detect fake news and trace how misinformation spreads through social networks. Built with a full-stack AI pipeline combining state-of-the-art NLP, Retrieval-Augmented Generation (RAG), epidemiological propagation modeling, and real-time monitoring, it delivers actionable intelligence at enterprise scale.

The platform evolved from an academic proof-of-concept into a scalable B2B SaaS product — featuring a FastAPI microservice backend, Dockerized deployment, Redis-based rate-limiting, and an enterprise analytics dashboard.

Core AI Pipeline

BERT & DeBERTa-v3 Classification

Uses mrm8488/bert-tiny-finetuned-fake-news-detection as the primary inference model with a DeBERTa-v3 ONNX Runtime upgrade path for sub-100ms production latency. Both models perform deep semantic classification of text.

Claim Extraction Engine

A custom NLP module (ClaimExtractor) identifies verifiable factual statements within submitted text using a scored heuristic pipeline. Each extracted claim is passed to the RAG pipeline for semantic fact-checking.

RAG Fact Verification (Vector DB)

Extracted claims are queries against a Knowledge Base using semantic similarity (Jaccard / Cosine). The system returns matched evidence from trusted sources (Reuters, AP News, WHO) with a human-readable AI RAG explanation.

Threat Assessment (Risk Scorer)

Computes a 0–100 Threat Score by combining: BERT prediction confidence, viral potential (R₀), emotional intensity sentiment, and extracted claim density — giving enterprise clients a single actionable danger metric.

SHAP Explainability (XAI)

SHapley Additive exPlanations quantify the marginal contribution of each word to the final verdict. The AI Truth Consultant report translates SHAP values into plain English — essential for enterprise trust and auditability.

SVM Baseline Model

A TF-IDF + Support Vector Machine model provides a secondary fast-path classification. Dual-consensus — where both BERT and SVM agree — significantly increases prediction confidence and serves as a cross-validation check.

Propagation & Network Analysis

SIR Epidemiological Model

Adapts the Susceptible–Infected–Recovered model to simulate viral spread of fake news across social networks. Calculates R₀ (basic reproduction number), peak day, and total estimated reach.

Graph Neural Network (GNN)

A next-generation GraphSAGE (Inductive GNN) scaffold is included to replace the static SIR model. GNNs learn propagation patterns from actual social graph topology, predicting cascade velocity and identifying echo-chamber bridge nodes.

Real-Time Feed Monitor

The /api/monitor/live endpoint simulates ingestion from RSS feeds, Twitter API, and Reddit streams. Each incoming headline is instantly scored by the AI pipeline and surfaced with its threat level.

How the Analysis Pipeline Works

  • Step 1 – Input Ingestion: Text, URL, or Extension context-menu selection is parsed and normalized.
  • Step 2 – BERT Inference: Pre-trained transformer model classifies content and returns confidence score.
  • Step 3 – SVM Cross-Validation: A secondary TF-IDF + SVM model provides a consensus check.
  • Step 4 – Claim Extraction: NLP module extracts verifiable facts from the article body.
  • Step 5 – RAG Fact Retrieval: Claims are semantically queried against the curated Knowledge Base Vector DB.
  • Step 6 – SIR Propagation Modeling: Estimates how fast and far the content would spread on social networks.
  • Step 7 – Risk Scoring: A composite 0–100 Threat Score is computed from all signals.
  • Step 8 – Report Generation: A PDF-exportable summary is produced with all insight layers.

Production Architecture

The platform is Dockerized and designed for Kubernetes deployment. The microservice stack consists of:

FastAPI Inference Service

Async ASGI server (Uvicorn) with Pydantic schema validation. Supports concurrent requests via asyncio without blocking inference threads.

Redis Rate Limiting

Atomic Redis pipeline enforces per-organization RPM (requests per minute) quotas based on subscription tier (Free / Pro / Enterprise).

Stripe Billing Integration

API Key Manager issues SHA-256 hashed developer keys. Billing tiers control access to premium features like RAG and custom fine-tuning.

Chrome Extension

Right-click any highlighted text on any website. The browser extension calls /api/extension/analyze and overlays the Threat verdict directly on the page.

Why It Matters

In an era of algorithmic amplification and social media echo chambers, misinformation spreads faster than corrections. This platform bridges the gap between academic NLP research and real-world operational deployment — giving journalists, platforms, financial intelligence teams, and governments a verified, explainable, and auditable tool for information integrity.

By making the invisible mechanics of misinformation propagation visible — from a single article to a viral network cascade — this system empowers faster, smarter, and evidence-backed responses to information crises.
